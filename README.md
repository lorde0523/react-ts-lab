# 🧪 React-TS-Lab

이곳은 **React**와 **TypeScript**를 처음부터 배우며 다양한 기능을 실험하고 기록하는 공간입니다.
기초 문법부터 작은 토이 프로젝트까지, 학습의 모든 과정을 단계별로 정리합니다.

## 🎯 학습 목표 (Goals)
- [ ] JavaScript(ES6+) 핵심 개념 복습
- [ ] TypeScript 기본 문법 및 타입 시스템 이해
- [ ] React의 동작 원리와 Hooks 완벽 적응
- [ ] TypeScript환경에서 React 컴포넌트 설계하기
- [ ] 혼자 힘으로 미니 프로젝트 3개 완성하기

## 📂 디렉토리 구조 (Structure)
학습 단계에 따라 폴더를 나누어 코드를 관리합니다.

📦 react-ts-lab
├── 📂 00-js-review       # 자바스크립트 핵심 문법 복습 (ES6+)
├── 📂 01-ts-basics       # 타입스크립트 기본 문법 연습
├── 📂 02-react-concepts  # 리액트 기초 (Props, State, Hooks)
├── 📂 03-mini-projects   # 실습 프로젝트 모음
│   ├── 📁 counter-app
│   └── 📁 todo-list
└── 📝 README.md          # 학습 기록

## 🛠 기술 스택 (Tech Stack)
- **Language**: TypeScript, JavaScript
- **Library**: React
- **Build Tool**: Vite
- **Package Manager**: npm
